# Sistema de Gestión de Inventario
## Descripción

Este proyecto corresponde a la evaluación del Módulo 3 de Python del curso SENCE.  
El programa permite registrar productos, consultar información, modificar datos, eliminar productos, controlar entradas y salidas de stock y calcular el valor total del inventario.

Funcionalidades

- Agregar productos.
- Mostrar todos los productos.
- Buscar productos por código.
- Modificar nombre, categoría o precio.
- Eliminar productos.
- Registrar entradas de stock.
- Registrar salidas de stock.
- Calcular el valor total del inventario.
- Mostrar categorías sin repetir.
- Mostrar el historial de movimientos.
- Validar los datos ingresados.

Estructuras utilizadas

- **Diccionario:** almacena los productos usando su código.
- **Lista:** guarda los movimientos del inventario.
- **Tupla:** contiene el nombre del sistema y el porcentaje de IVA.
- **Conjunto:** almacena las categorías sin duplicados.
